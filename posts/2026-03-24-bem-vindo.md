# Welcome

This blog supports **Markdown**, diagrams, math formulas, and function graphs.

## Diagrams with Mermaid

Sequence diagram:

```mermaid
sequenceDiagram
    participant Client
    participant Server
    participant Database
    Client->>Server: POST /api/data
    Server->>Database: INSERT query
    Database-->>Server: OK
    Server-->>Client: 201 Created
```

Use case:

```mermaid
graph LR
    User((User))
    User --> Login[Login]
    User --> ViewPosts[View Posts]
    User --> CreatePost[Create Post]
    Admin((Admin))
    Admin --> ManageUsers[Manage Users]
    Admin --> CreatePost
```

## Code

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```

## How to use

1. Create a `.md` file in `posts/` with front matter:
   ```yaml
   ---
   title: My Post
   date: 2026-03-24
   ---
   ```
2. Push to `main`
3. CI generates `posts.json` automatically
