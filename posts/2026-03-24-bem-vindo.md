## Código

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```

## Como usar

   ```yaml
   ---
   title: Meu Post
   date: 2026-03-24
   ---
   ```
2. Faça push para `main`
3. O CI gera o `posts.json` automaticamente
